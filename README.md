# 🕊️ The Economy of Truth

> **Récompenser la Vérité. Déraciner le Mensonge.**  
> Un protocole anonyme et décentralisé pour les révélateurs de vérité.

---

## 🧭 Constat

Le monde repose sur des vérités cachées.  
Les crimes d’État, la corruption systémique, les ententes occultes entre entreprises et gouvernements, les dérives médiatiques…  
Tout cela persiste parce que **le silence est rentable, et la vérité coûte cher**.

👉 Ce projet part d’un postulat simple :  
> _« Si dire la vérité ruinait moins que se taire, le monde serait différent. »_

---

## 🎯 Objectif

Créer une **infrastructure décentralisée** où :

- Toute personne peut **soumettre une révélation** en restant anonyme.
- La **véracité** est évaluée collectivement via un système de **DAO + réputation + preuve**.
- Les révélations **impactantes et vérifiées** sont **récompensées** automatiquement via des **smart contracts**.

---

## 🧱 Architecture Technique

### 🔐 Anonymat & Sécurité
- Accès via **Tor**, **Tails**, ou autres systèmes anonymisants.
- Soumission chiffrée côté client, métadonnées nettoyées.
- Stockage des données via **IPFS**, **Arweave**, **Skynet**, etc.
- Hash de preuve ancré dans une **blockchain publique**.
- Identité masquée, traçabilité **de l’information**, pas des individus.

### ⚙️ Smart Contracts
- Token natif : `TRUTH` ou `EOT` (Economy of Truth Token).
- Contrats autonomes pour :
  - Verrouillage de stake
  - Validation DAO
  - Redistribution automatique des fonds selon résultat

### 🗳️ DAO (Gouvernance Décentralisée)
- Réputation évolutive des validateurs
- Votes pondérés par crédibilité et historique
- Pools de fonds thématiques créés et gérés collectivement

---

## 🔁 Processus de Révélation

1. **Soumission**
   - Un utilisateur stake une quantité de tokens TRUTH pour publier une révélation.
   - La révélation est hachée, déposée de façon anonyme et distribuée.

2. **Validation Communautaire**
   - DAO d'experts/validateurs examine les preuves.
   - Prise en compte de la **gravité, vérifiabilité, impact démocratique**.

3. **Résultat du vote**
   | Verdict                   | Action                                                                 |
   |--------------------------|------------------------------------------------------------------------|
   | ✅ Vérité vérifiée        | Stake rendu + Bonus (selon impact)                                     |
   | ⚠️ Vérité mineure         | Stake rendu, pas de bonus ou bonus faible                              |
   | ❌ Mensonge/manipulation  | Stake brûlé ou redistribué, réputation réduite                         |

4. **Publication**
   - Le document devient public (si choix de l’auteur) ou reste distribué anonymement.
   - Il est archivé de manière immuable.

## 💸 Mécanismes d’Incitation

> **Proof of Truth (PoT)** — Un nouveau consensus basé sur la **véracité**.

- **Staking** pour poster (responsabilisation).
- **Récompense** pour ceux qui disent la vérité.
- **Pénalité** pour ceux qui mentent ou manipulent.

## 📊 Pools Thématiques (DAO)

Chaque DAO peut créer un pool thématique avec un bounty :

| Pool                          | Description                                          |
|------------------------------|------------------------------------------------------|
| 🌍 Crimes environnementaux   | Révélations liées à la destruction de la planète     |
| 🏛️ Corruption publique       | Contrats opaques, magouilles politiques              |
| 📺 Mensonges médiatiques     | Manipulation de masse, désinformation institutionnalisée |
| 🧪 Pression sur la recherche | Censure académique, résultats altérés                |
| 🔬 Industrie & collusion     | Accords secrets, conflits d’intérêts pharmas/tech   |

## 💥 Cas d’usage concrets

- **Un journaliste** dévoile un contrat secret entre une chaîne publique et une entreprise d’armement → DAO + preuves solides → Récompense élevée.
- **Un employé IT** révèle la manipulation d’un algorithme électoral → Audit technique + votes → Bonus majeur + badge DAO.
- **Un étudiant** montre la censure d’un mémoire sur les pesticides → Récompense modeste, mais préservation de l’anonymat.

## 🧠 Éthique & Valeurs

- **⚖️ Éthique avant tout** : Toute révélation doit servir le bien commun.
- **🔐 Anonymat comme droit** : Aucun identifiant stocké, full OpSec.
- **📣 Révélation comme acte citoyen** : Encourager la transparence.
- **⛓️ Décentralisation comme rempart** : Personne ne peut supprimer ce qui est distribué.
- **🧠 Vérité comme monnaie** : C’est l’impact et la vérifiabilité qui comptent.

## 🛠️ Stack technique recommandée

- **Frontend** : React, Svelte, ou static HTML (serveur Tor/Tails)
- **Backend** : Smart contracts Solidity (Ethereum, Arbitrum, etc.)
- **Storage** : IPFS, Filecoin, Arweave, Skynet
- **Gouvernance** : DAOstack, Snapshot, Gnosis Safe
- **Tokenomics** : ERC-20 pour TRUTH, staking via ERC-4626 vaults

## 🔓 Sécurité & OPSEC

- Tous les contributeurs doivent suivre un **guide de sécurité OpSec** :
  - Clé GPG
  - No JS, navigateur sandboxé
  - VPN + Tor combinés
  - Machines airgap pour manipuler les fichiers originaux
  - Utilisation de Metamask via bridge .onion
