🤝 CONTRIBUTING TO ECONOMY OF TRUTH


---

🇬🇧 EN | English

Thank you for considering contributing to the Economy of Truth — a decentralized, ownerless protocol to protect, verify, and reward truth.

Whether you're a developer, designer, hacktivist, or a concerned citizen, your contribution matters.

📌 How to Contribute

1. Fork this repository.


2. Clone your fork locally:

git clone https://github.com/yourusername/Economy-Of-Truth.git


3. Create a new branch:

git checkout -b feature/my-contribution


4. Make your changes.


5. Submit a Pull Request with clear description and context.



💡 What You Can Help With

Web development (frontend/backend)

Blockchain & crypto development (Solidity / Ink!)

Security & anonymization tools

DAO & governance logic

Multilingual interface & translation

Documentation, community building, design


✅ Guidelines

Prioritize ethics, privacy, and decentralization.

Comment and document your code.

Write meaningful commit messages.

Never push directly to main.


Together, we build a protocol that outlives fear.
